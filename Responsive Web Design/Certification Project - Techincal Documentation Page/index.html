Introduction

JavaScript is a cross-platform, object-oriented scripting language.

It can be used in web browsers and on servers.

JavaScript provides flexibility to work with dynamic content.

console.log('Hello, world!');

You can extend core JavaScript with additional libraries.

  • Client-side JavaScript works in browsers.
  • Server-side JavaScript works on servers (Node.js).
Variables

Variables store data values in JavaScript.

There are three ways to declare variables:

  • var x = 42;
  • let y = 'hello';
  • const z = true;

Variable names must follow specific rules.

JavaScript is case-sensitive, so 'varName' and 'varname' are different.

Variables can hold any type of data.

Data Types

JavaScript supports several data types.

  • String
  • Number
  • Boolean
  • Object
  • Array

Strings represent text data.

Numbers can be integers or floats.

let count = 5;

Booleans are true or false.

Objects store key-value pairs.

let person = {name: 'John', age: 30};
Statements

Statements are the building blocks of JavaScript programs.

Each statement performs an action.

Statements end with a semicolon.

  • Declaration Statements
  • Assignment Statements
  • Control Flow Statements

Control flow statements include:

  • if (condition) { }
  • for (let i = 0; i < 5; i++) { }
Functions

Functions are reusable blocks of code.

They can take arguments and return values.

function greet(name) { return 'Hello ' + name; }

Functions can be declared or expressed.

Functions can also be anonymous or arrow functions.

const add = (a, b) => a + b;

Functions are first-class citizens in JavaScript.
